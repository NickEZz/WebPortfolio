About Point Cloud Viewer

PointCloud Viewer

It is made using C# and OpenTK library, The component renderes point cloud from .CVS file and shows it inside winforms compontent. There is multiple ways to control what is showing, and all opf the code has been coded so you can later change everyting from another scrip for example color, size, render mode or load pointcloud from cloud or sensor directly
